OncoOmics in analysis will provide users to perform omics and network analysis based on data selection. The users can perform analysis about:

  • Gene Expression analysis/profiling
  • Mass Spectrometry analysis
  • Metabolic Profiling/analysis
  • Biomarker analysis
  • Text Mining analysis
  • Pathway analysis/Gene-Drug Network/Protein-Protein Network
  • Clinical data analysis
  • Pharmoco-genomics data analysis
